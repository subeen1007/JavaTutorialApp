※제어문-반복문 <반복문> ●반복문을 쓰는 이유 -같은 처리 과정을 되풀이하는 것이 필요하기 때문이다. 학생 30명의 평균 성적을 구하려면 같은 과정을 30번 반복하여야 한다. ●반복문의 종류 - while문 - do-while문 - for문 ●while문 -주어진 조건이 만족되는 동안 문장들을 반복 실행한다. -형식: while(조건식) { //조건식은 반복을 하는 조건이다. 참이면 반복을 계속한다. 문장1; //반복되는 문장이다. 문장2; ... } ●do-while문 -일단 문장을 실행한뒤, 주어진 조건이 만족되는 동안 문장들을 반복 실행한다. -형식: do{ //처음에는 조건식과 상관없이 일단 문장을 실행시킨다. 문장1; //반복되는 문장이다. 문장2; ... } while(조건식) // 조건식이 참이면 계속 문장을 반복, 거짓이면 반복을 종료한다. ●for문 형식: for(초기화; 종료조건; 증감식){ 반복문장; } -EX) for(i = 0; i < 5; i++) { System.out.print("Hello"); } 결과: Hello Hello Hello Hello Hello ●중첩 반복문 형식: for (i = 0; i < 5; i++) { //외부 반복문 for (k = 0; k < 5; k++) { //내부 반복문 반복문장; } }
