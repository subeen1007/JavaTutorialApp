※배열 <배열의 필요성> ●배열이 필요한 이유 -예를 들어서 학생이 10명이 있고 이들의 성적의 평균을 계산한다고 가정하자. 학생이 10명이므로 10개의 변수가 필요하다. int s0, s1, s2, s3, s4, s5, s6, s7, s8, s9; 하지만 만약 학생이 100명이라면 어떻게 해야 하는가? int s0, s1, s2, s3, s4, s5, s6, s7, s8, s9; ..., s99; ●배열이란 -배열(array): 동일한 타입의 변수들의 모임 <배열의 선언과 사용> ●배열을 만드는 절차 1.먼저 배열 참조 변수부터 선언 ex) int[] s; 2.배열을 new 연산자를 사용하여서 생성 ex) s = new int[10]; //[10]은 배열의 크기이다. -이때 s를 참조변수라 한다. ●배열의 인덱스 -다음과 같은 배열을 가정하자. int[] s = new int[10]; → s[0], s[1], s[2], s[3], s[4], s[5], s[6], s[7], s[8], s[9] -배열 요소에는 번호가 붙어 있는데 이것을 인덱스(index)라고 부른다. -첫 번째 요소의 번호는 0이고, 마지막 요소의 번호는 9가 된다. ●인덱스를 통한 요소의 접근 -배열은 변수들이 모인 것이니, 배열을 이루고 있는 배열 요소는 하나의 변수로 생각하면 된다. -배열의 첫 번째 요소에 80을 저장하려면 다음과 같이 한다. s[0] = 80; s[0], s[1], s[2], s[3], s[4], s[5], s[6], s[7], s[8], s[9] ↑ 80 ●배열의 초기화 -EX) int[] scores = {10, 20, 30, 40, 50}; //scores은 변수이름 ●문자열 배열 -String[]을 사용한다. -EX) String[] toppings = {"Pepperoni", "Mushrooms", "Onions", "Sausage", "Bacon"}; <고급 배열> ●무명 배열 -자바에서는 배열의 이름을 지정하지 않고, 단순히 초기값만으로 배열을 생성시킬 수있다. -무명 배열(anonymous arrays)은 즉시 배열을 만들어서 함수의 인수로 전달하고자 할 때 많이 사용된다 -형식: new int[] {1, 2, 3, 4, 5, 6, 7, 8} ●for-each -배열의 원소를 하나씩 꺼낸다. -EX) String[] members ={"A", "B", "C"}; // 문자열 배열인 members 변수를 만든다. for(String e : members) { //배열members의 값을 변수e에 담아서 원소를 하나씩 꺼낸다. System.out.println(e); } 결과: A B C ●배열의 복사 -배열 참조 변수의 복사 int [] list = {10, 20, 30, 40}; int [] numbers = list; -한 배열의 모든 값을 다른 배열로 복사하고 싶다면 Arrays 클래스의 copyOf() 메소드를 사용 -EX) int [] list_copy = Arrays.copyOf(list, list.length); <배열 정렬> ●배열 정렬 -배열에 저장된 숫자를 크기 순으로 정렬하려면 Arrays.sort() 사용 EX) int[] a =new int[100]; a[0] = 32; a[1] = 21; ... Arrays.sort(a); <2차원 배열> ●2차원 배열 -형식: int[][] s = new int[3][5]; //[][]는 2차원 배열을 나타낸다. 3은 행의 개수, 5는 열의 개수이다. 열→ s[0][0] s[0][1] s[0][2] s[0][3] s[0][4] 행 s[1][0] s[1][1] s[1][2] s[1][3] s[1][4] ↓ s[2][0] s[2][1] s[2][2] s[2][3] s[2][4] ●2차원 배열의 초기화 int[][] testArray = { {10, 20, 30}, {40, 50, 60}, {70, 80, 90} };
